# S4 - HTTP

[![Slack][slack-badge]][slack-url]

HTTP standards for Swift.

## Installation

- Add `S4` to your `Package.swift`

```swift
import PackageDescription

let package = Package(
    dependencies: [
        .Package(url: "https://github.com/open-swift/S4.git", majorVersion: 0, minor: 6)
    ]
)
```
